PDF to Text Extractor

Upload a text-based PDF (up to 10 MB) and extract all selectable text into a single editable box. For scanned PDFs, use the Image → Text scanner.

1. Upload PDF

Supports standard PDFs up to 10 MB. Larger files will be rejected before processing.

Text is extracted locally in your browser using pdf.js. The file is not uploaded to any server.
Big PDFs with many pages may take longer because each page is processed once. [web:344]

2. Extracted text

After extraction, edit or copy the text below.

If your PDF is just photos/scans with no selectable text, use the Text Scanner tool instead. [web:356]
